# 🔍 篩查技術與早期發現

主動篩查是早期發現癌症、降低死亡率最有效的手段。以下是目前醫學界公認的關鍵篩查建議：

### 1. 影像學篩查 (最成熟的手段)
* **肺癌篩查 (LDCT)**：
    * **建議對象**：45 歲以上的高危人士（如吸菸者或有家族病史）。
    * **技術**：**低劑量電腦掃描 (LDCT)**。它比傳統 X 光更精確，能發現直徑極小的病變，如「磨玻璃影 (GGO)」。
* **大腸癌篩查**：
    * **建議對象**：50 歲以上人士。
    * **方法**：定期進行大腸鏡檢查，若發現息肉可即時移除，防止其演變成惡性腫瘤。



### 2. 血液檢測新趨勢 (液體切片)
* **EB 病毒 DNA 篩查**：這是香港在國際上的重要突破，透過抽血檢測 EB 病毒 DNA，能有效篩查早期**鼻咽癌**，顯著提高生存率。
* **ctDNA (循環腫瘤 DNA)**：透過血液捕捉癌細胞釋放的 DNA 片段。雖然目前在追蹤晚期癌症復發上較為成熟，但在早期篩查領域已顯示出巨大潛力。
* **注意**：單純的循環腫瘤細胞 (CTC) 檢測在早期癌症中可能敏感度不足，通常需配合其他檢查。

### 3. 遺傳與基因因素
如果一個人在短時間內多次患癌，或家族中多位近親患有同類癌症，可能與 **BRCA** 等基因突變有關。
* **應對方案**：此類高危族群應在醫生建議下提前開始篩查，而非等待 50 歲。

### 4. 常見迷思：良性能變惡性嗎？
良性腫瘤通常生長緩慢且有包膜，不易擴散。但某些「癌前病變」如肺部的磨玻璃影，雖然發現時可能不是惡性，但若不及時處理，未來仍有演變風險。

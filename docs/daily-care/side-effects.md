# 化療與電療副作用管理

治療期間的副作用因人而異，以下是常見問題的護理建議：

### 1. 噁心與嘔吐 (Nausea)
* **護理**: 少量多餐，避免進食過油或味道太重的食物。
* **建議**: 依照醫生指示服用止嘔藥，不要等到想嘔才吃。

### 2. 脫髮 (Hair Loss)
* **心理準備**: 通常在第一次化療後 2-3 週開始。
* **護理**: 使用溫和的洗髮水，考慮提前準備假髮或頭巾。

### 3. 皮膚乾燥與發炎
* **電療護理**: 保持受檢部位皮膚乾爽，避免使用含酒精的護膚品。
* **建議**: 諮詢醫療團隊有關專用的電療潤膚膏。

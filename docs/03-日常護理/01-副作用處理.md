# 化療與電療常見副作用處理

化學治療、放射治療及部分標靶或免疫治療，可能會影響身體中正常而分裂較快的細胞，例如口腔黏膜、腸胃道、骨髓、皮膚及毛囊。因此，治療期間出現噁心、口腔痛、腹瀉、疲倦、脫髮或感染風險上升，並不罕見。

副作用管理的目標不是「硬撐」，而是及早預防、及早處理，讓療程盡量順利完成。以下內容只供日常護理參考，不能取代醫生、腫瘤科護士或營養師的個別建議。

## 何時要立即求醫？

如果出現以下情況，不要等覆診，應盡快聯絡治療團隊或到急症室：

- 發燒，特別是體溫達 38°C 或以上，或伴隨發冷、寒顫。
- 一小時內嘔吐三次或以上，或完全無法飲水、進食、服藥。
- 嚴重腹瀉、血便、黑便、劇烈腹痛或有脫水跡象。
- 出血超過 10 分鐘仍未能止住，或出現血尿、大片瘀斑、嚴重頭痛。
- 口腔潰瘍嚴重至不能吞嚥，或同時有發燒、口腔白斑、膿液。
- 放射治療位置出現水泡、滲液、異味、紅腫熱痛加劇。
- 呼吸困難、胸痛、意識混亂、突然乏力或步態不穩。

## 噁心與嘔吐

化療引起的噁心嘔吐可以分為急性、延遲性及預期性。最重要的原則是「預防勝於補救」：如醫生已處方止嘔藥，應按時服用，不要等到嘔吐後才服。

| 做法 | 建議 |
| --- | --- |
| 進食方式 | 少量多餐，每日 5 至 6 餐；清晨或活動前可先吃蘇打餅、白吐司等乾身食物。 |
| 食物溫度 | 選擇室溫或偏涼食物，例如冷麵、水果、涼拌豆腐；熱食氣味較易誘發噁心。 |
| 味道調節 | 可用酸梅、檸檬汁、陳皮、薑味食物改善口淡或噁心，但胃酸倒流者要小心。 |
| 飲水時間 | 盡量在兩餐之間補水；用餐時避免一次飲太多，以免胃脹。 |
| 避免食物 | 暫時避開油炸、高脂、辛辣、過甜及氣味濃烈的食物。 |
| 環境 | 在通風位置進食，避開油煙、香煙、香水或其他強烈氣味。 |

如果剛嘔吐完，先漱口，休息 15 至 30 分鐘，再由少量清水、米湯、白粥或去油清湯開始。不要勉強一次吃完整碗，能每 1 至 2 小時吃幾口，也算是進步。

## 腹瀉與腸胃不適

治療期間腹瀉可能來自化療、放療、標靶藥、免疫治療、抗生素或感染。處理方向取決於原因，所以若腹瀉持續、愈來愈嚴重，或伴隨發燒、血便、腹痛，應通知醫護人員。

日常可先採取以下做法：

- 飲食以清淡、低油、低纖維為主，例如白粥、米湯、白飯、麵包、去油清湯、蒸蛋。
- 暫停油炸食物，例如炸兩、油條、薯條，因為高脂肪會加重腸胃負擔。
- 少量多餐，不要一次吃太飽。
- 補充水分與電解質，可選水、口服補液、清湯；運動飲品可偶爾補充，但糖分高，宜少量。
- 暫時避免酒精、咖啡、奶類、辛辣食物、生冷食物及高纖食物。
- 排便後用溫水輕柔清潔肛門位置，避免用力擦拭。

未經醫生指示，不建議自行長期使用止瀉藥，尤其是懷疑感染或免疫治療相關腸炎時。

## 口腔黏膜炎與口腔護理

口腔黏膜炎常見於化療及頭頸部放射治療，通常在治療後 7 至 14 天出現。可能表現為口乾、紅腫、疼痛、潰瘍、吞嚥困難或口腔白斑。

### 預防與日常護理

- 治療前如時間許可，先做牙科檢查，處理蛀牙、牙周病或不合適假牙。
- 使用軟毛或極軟毛牙刷，可先用溫水浸軟刷毛。
- 每日照鏡檢查口腔，包括舌頭、牙齦、面頰內側及喉嚨附近。
- 避免含酒精漱口水，因為會刺激及令黏膜更乾。
- 可按醫護建議使用生理鹽水或梳打水漱口；如有白斑或疑似感染，需由醫生評估是否需要抗黴菌藥。
- 避免煙酒、辣、酸、粗硬、過熱食物，例如薯片、堅果、火鍋熱湯。

| 情況 | 飲食與護理方向 |
| --- | --- |
| 輕微紅腫、乾痛 | 保持清潔，多漱口，選擇軟身、濕潤食物。 |
| 有潰瘍但仍能進食 | 改為半流質或軟餐，例如粥、蒸蛋、豆腐、魚肉、營養奶。 |
| 嚴重疼痛，只能飲流質 | 聯絡醫護，可能需要止痛漱口水、止痛藥、營養支援。 |
| 無法吞嚥或伴隨發燒 | 應盡快求醫，需排除感染及脫水。 |

## 白血球低與感染風險

化療可能令骨髓暫時受抑制，白血球，尤其是中性粒細胞下降。通常在治療後約 7 至 10 天附近較低，但每種藥物不同，應以醫護提供的驗血時間表為準。

降低感染風險的重點：

- 經常洗手，尤其是進食前、如廁後、外出回家後。
- 避免接觸發燒、咳嗽、腹瀉或有傳染病症狀的人。
- 人多擠迫或空氣不流通地方可戴外科口罩。
- 食物要煮熟，避免生魚片、半熟蛋、生蠔、未洗乾淨的生菜沙律、未剝皮水果。
- 皮膚如有傷口，要保持清潔乾爽，若紅腫熱痛或流膿要通知醫護。
- 不要自行服用退燒藥掩蓋發燒，除非醫生已教你何時可用。

發燒可能是中性粒細胞低下感染的唯一訊號，即使沒有咳嗽或喉嚨痛，也要認真處理。

## 血小板低與出血預防

血小板低時，身體較容易瘀青或出血。若醫護告知你的血小板偏低，可注意以下事項：

- 使用軟毛牙刷，避免大力刷牙。
- 避免剔牙、用力擤鼻或挖鼻。
- 使用電動刮鬍刀，避免刀片刮傷。
- 避免碰撞、跌倒及高風險運動。
- 不要自行服用阿士匹靈或非類固醇消炎止痛藥，例如 ibuprofen、naproxen，除非醫生批准。
- 避免自行使用灌腸、肛塞或量肛溫。

如出現黑便、血尿、吐血、持續流鼻血、嚴重頭痛或視力改變，應立即求醫。

## 脫髮與頭皮護理

不是所有治療都會脫髮，但部分化療會在第一次療程後約 2 至 3 星期開始掉髮。多數情況下，療程完成後 3 至 6 個月會逐漸重新長出，但髮質或顏色可能暫時改變。

護理建議：

- 使用溫和、無香味洗髮產品。
- 避免染髮、電髮、漂髮及高溫吹風。
- 可預先剪短頭髮，減少掉髮時的心理衝擊。
- 外出時保護頭皮，避免曝曬；可使用帽、頭巾或假髮。
- 如醫院有提供頭皮冷卻服務，可向治療團隊查詢是否適合自己。

## 放射治療皮膚護理

放射治療造成的皮膚反應通常是累積性的，可能在療程數週後逐漸出現，包括泛紅、乾燥、痕癢、脫皮，嚴重時可有水泡或滲液。

照射區域護理原則：

| 項目 | 建議 |
| --- | --- |
| 清潔 | 用溫水輕柔清洗，避免熱水、刷洗、浸浴及磨砂產品。 |
| 衣物 | 穿寬鬆、柔軟、棉質衣物，減少摩擦。 |
| 護膚品 | 未經醫護同意，不要在照射前塗乳液、藥膏、香水、爽身粉或油性產品。 |
| 定位線 | 不要自行擦走或重新描畫治療定位標記。 |
| 防曬 | 避免陽光直射照射區域，外出以衣物遮蓋。 |
| 觀察 | 每日留意水泡、濕性脫皮、滲液、異味、紅腫熱痛。 |

如皮膚完整，醫護可能會建議使用指定保濕產品。若已有破損、滲液或水泡，不要自行敷藥，應由治療團隊評估敷料及傷口護理方式。

## 手足症候群與手腳麻痺

部分化療、標靶藥或口服抗癌藥可引起手足症候群，常見症狀包括手掌、腳底紅、腫、熱、痛、脫皮或裂開。另一些藥物可造成周邊神經病變，出現手腳麻痺、刺痛、怕冷或扣鈕困難。

日常照顧：

- 保持手腳滋潤，可向醫護查詢是否適合使用含尿素的潤膚膏。
- 避免熱水浸手腳、桑拿、長時間行走、跑步或手部用力摩擦。
- 做家務時戴手套，但避免太緊或悶熱。
- 穿柔軟合腳的鞋襪，避免赤腳走路。
- 手腳麻痺時要小心熱水、刀具及跌倒風險。
- 若麻痺影響走路、拿東西或日常生活，應告訴醫生，可能需要調整藥物或安排治療。

維他命 B 群或其他補充品不應自行大量服用，因為可能與治療或其他藥物互相影響，應先問醫生或藥劑師。

## 疲倦與體力管理

癌因性疲憊不是單純「睡不夠」，即使休息後仍可能覺得無力。它可能與治療、貧血、營養不足、睡眠、情緒壓力或疼痛有關。

實用策略：

- 把最重要的事情安排在一天中精神最好的時段。
- 家務和外出分段完成，中間安排休息。
- 小睡可以，但盡量避免白天睡太久，以免影響夜晚睡眠。
- 體力許可時維持輕量活動，例如室內慢行、伸展、短距離散步。
- 疲倦時休息是正確的；醒後如體力稍回復，可嘗試輕微走動，有助腸胃蠕動及減少長時間臥床的不適。

如果疲倦突然惡化、伴隨氣促、心悸、頭暈、胸痛或發燒，應盡快求醫。

## 營養與食物交互作用

治療期間，足夠熱量和蛋白質有助維持體重、修復組織和支撐免疫功能。若食慾差，可先追求「吃得到、吃得下」，再慢慢改善質量。

增加營養密度的方法：

- 粥或湯粉中加入蛋花、豆腐、魚肉、碎肉。
- 用牛奶、豆漿或營養奶代替清水沖調部分食物。
- 小份量多次進食，身邊放容易入口的小食。
- 口淡或有金屬味時，可試用膠匙、竹筷，或用少量檸檬水改善味覺。

治療期間要特別留意以下食物或補充品：

| 類別 | 原因與建議 |
| --- | --- |
| 西柚、柚子、葡萄柚 | 可影響部分藥物代謝，增加副作用或改變藥效；是否需要完全避免，應問醫生或藥劑師。 |
| 聖約翰草 | 可能降低部分抗癌藥血藥濃度，治療期間一般不建議使用。 |
| 酒精 | 可能刺激黏膜、影響肝臟代謝及增加脫水風險，治療期間宜避免。 |
| 高劑量保健品 | 抗氧化劑、草藥、濃縮萃取物可能與治療互相影響，服用前應先查詢醫護。 |
| 天然豆類食品 | 豆腐、豆漿、黃豆等可作為蛋白來源；若有特定癌種或荷爾蒙治療疑問，應向醫生或營養師確認。 |

## 壓力與呼吸練習

焦慮和壓力會放大噁心、疼痛與疲倦感。以下方法可作為副作用發作時的輔助工具：

- 腹式呼吸：一手放胸前，一手放腹部，慢慢吸氣，感受腹部隆起，再慢慢呼氣。
- 箱式呼吸：吸氣 4 秒、停 4 秒、呼氣 4 秒、停 4 秒，循環數次。
- 分散注意力：聽音樂、短時間散步、看輕鬆影片、與可信任的人傾談。

若情緒長期低落、失眠、恐慌或有傷害自己的想法，應告訴醫護人員，讓他們協助轉介心理支援、社工或相關服務。

## 簡易照顧者提醒

照顧者可以幫忙記錄每日體溫、進食量、飲水量、排便次數、嘔吐次數、疼痛位置和用藥時間。這些資料在覆診或致電醫院時很有用。

當患者不想吃東西時，先避免責備或強迫。可以改問：「現在想試兩口粥、幾啖湯，還是先漱口休息？」把選擇變小，通常比勉強吃完整餐更可行。

# 癌症病人及家屬支援組織

香港有多個非牟利機構為病友提供免費支援，建議善用：

### 1. 香港癌症基金會 (Hong Kong Cancer Fund)
* **服務**: 提供專業諮詢、心理輔導、營養建議及各類復康工作坊。
* **地點**: 中環、旺角、觀塘等設有服務中心。

### 2. 銘琪癌症關顧中心 (Maggie's Centre)
* **特點**: 位於屯門醫院附近，環境優美，提供放鬆身心的空間與小組支援。

### 3. 醫管局藥物資助 (撒瑪利亞基金)
* **對象**: 有經濟困難但需要使用自費特效藥物（如部分標靶藥）的病人。
* **申請**: 需聯繫醫院醫務社會工作者進行入息審查。

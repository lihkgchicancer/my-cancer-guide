# 👥 同路人：病友互助組織

在抗癌路上，與有相似經歷的「同路人」交流，能提供極大的心理慰藉。

### 1. 智友站 (Smart Patient)
這是醫院管理局建立的資源平台，整合了全港不同類型的病友組織。
* **功能**：按癌症類別（如：乳癌、大腸癌、肺癌）搜尋對應的互助組織。
* **資源**：提供自我護理手冊與家屬照顧者指南。

### 2. 同路人組織的價值
參與這些組織可以讓你：
* **經驗分享**：獲取有關副作用護理、日常飲食的實戰經驗。
* **情緒互助**：與明白你處境的人傾訴，減輕焦慮感。
* **資訊更新**：定期舉辦講座，邀請醫生講解最新治療趨勢。

> **如何尋找？**
> 你可以直接前往「智友站」網頁，或詢問醫院門診的資源中心。大多數組織如「乳青會」、「青雲社」等，均歡迎新確診者加入。

# 新確診癌症後 14 日行動清單

剛聽到「癌症」兩個字，很多人會立即失去方向。這份清單不是要你在 14 日內完成所有治療，而是幫你在最混亂的時間，先把最重要的資料、排期和決策基礎建立好。

請記住：癌症治療通常是團隊決定，不是靠一天內做完所有事。你現在最需要的是準確診斷、完整紀錄、清楚下一步。

## 第 0 至 1 日：先確認「我手上有甚麼」

請建立一個實體文件夾和一個手機/電腦資料夾，放入以下資料：

- 身份證副本、聯絡電話、緊急聯絡人。
- 所有化驗、掃描、內窺鏡、病理報告。
- 影像光碟或 USB，最好是 DICOM 格式。
- 私家醫生或公立醫院的轉介信。
- 目前服用中的藥物、保健品、中藥清單。
- 過敏紀錄、長期病、以往手術紀錄。
- 保險保單、醫療卡、公司醫療福利資料。

如果你只收到口頭通知，請盡快向診所或醫院索取正式書面報告。

## 第 1 至 3 日：問清楚診斷是否已完整

很多人以為「發現腫瘤」就等於已經可以治療，但治療前通常仍要確認幾件事：

| 要確認的事 | 為甚麼重要 |
| --- | --- |
| 癌症種類 | 不同癌種治療完全不同，例如肺癌、乳癌、淋巴癌不是同一套方案。 |
| 病理類型 | 例如腺癌、鱗狀細胞癌、小細胞癌、淋巴瘤等。 |
| 分期 | 決定治療目標是根治、控制病情還是紓緩症狀。 |
| biomarker / 基因檢測 | 影響標靶藥、免疫治療或臨床試驗選擇。 |
| 身體狀態 | 心肺功能、肝腎功能、營養、活動能力都會影響治療承受力。 |

你可以直接問醫生：「我現在的診斷已經足夠決定治療嗎？還欠哪一項檢查？」

## 第 1 至 5 日：同時處理公立與私家選項

不要等其中一邊完全完成才做另一邊。比較穩妥的做法是同步建立選項。

### 公立醫院

- 向私家醫生或普通科醫生索取轉介信。
- 盡快把轉介信交到所屬聯網或相關專科門診。
- 保留遞交日期、收據、電話紀錄和預約信。
- 如病情變差，不要只等門診，可到急症室。

### 私家醫療

- 若有保險，先問清楚門診、住院、日間治療、檢查和藥物保障。
- 如要見私家腫瘤科，帶齊所有報告和影像。
- 先問診斷和分期是否完整，不要急於即日決定昂貴治療。
- 要求清楚列出治療方案、預計次數、費用範圍和可能副作用。

## 第 3 至 7 日：補齊關鍵檢查

不同癌種需要的檢查不同，但常見包括：

- CT、MRI、PET-CT、骨掃描或超聲波，用於分期。
- 內窺鏡、支氣管鏡、穿刺或手術切片，用於取得組織。
- 病理覆檢，確認癌症類型。
- 基因檢測或 biomarker，例如 EGFR、ALK、KRAS、BRAF、HER2、PD-L1、MSI。
- 血液檢查，包括血球、肝腎功能、凝血、腫瘤指標。

如果你在私家完成檢查，記得索取正式報告和影像 DICOM。不要只拍手機相。

## 第 5 至 10 日：整理「病情摘要」

你可以用以下格式寫一頁病情摘要，覆診時交給醫生：

```text
姓名：
年齡：
確診日期：
癌症種類：
病理結果：
目前分期：
已知轉移位置：
已完成檢查：
未完成檢查：
已知 biomarker / 基因結果：
目前症狀：
現有長期病：
正在服用藥物：
保險/財政限制：
最想問醫生的 3 個問題：
```

癌症網上資源中心也建議病人建立個人健康紀錄，包括診斷日期、檢驗結果、癌症種類、分期、治療資料、副作用和醫療團隊聯絡資料。

## 第 7 至 14 日：決定下一步治療路線

當診斷和分期比較清楚後，請和主診醫生確認：

- 治療目標是根治、降低復發、控制病情，還是改善症狀？
- 是否需要多專科會議或第二意見？
- 第一個治療應該是手術、化療、電療、標靶、免疫治療，還是先處理併發症？
- 如果等公立排期，等待多久仍屬合理？
- 如果先私家治療，之後能否順利銜接公立？
- 有沒有需要及早找醫務社工、營養師、心理支援或病人資源中心？

## 每次覆診前要準備

- 寫下最近症狀：發燒、痛楚、胃口、體重、大小便、氣促、出血、麻痺。
- 帶齊最新報告和影像。
- 把問題按重要程度排序，先問最影響決策的三條。
- 帶一位家人或朋友一起聽，負責記錄。
- 問清楚下一次覆診、檢查、治療日期和聯絡方法。

## 紅色警戒：不要只等排期

如出現以下情況，應盡快求醫：

- 發燒、寒顫、持續嘔吐或不能飲水。
- 氣促、胸痛、突然嚴重頭痛、神志不清。
- 大量出血、黑便、血尿。
- 劇烈腹痛、腸塞症狀、黃疸。
- 癱瘓、抽搐、突然無力、大小便失禁。
- 痛楚失控，或藥物無法紓緩。

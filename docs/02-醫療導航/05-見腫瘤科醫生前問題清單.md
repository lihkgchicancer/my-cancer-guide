# 見腫瘤科醫生前問題清單

腫瘤科初診通常時間有限。最有效的方法，是覆診前先把問題寫好，並按「會影響治療決定」的程度排序。

你不需要一次問完所有問題。請先問最關鍵的三類：診斷是否完整、治療目標是甚麼、下一步要做甚麼。

## 初診必問

- 我的癌症診斷是否已經確定？癌症種類和病理類型是甚麼？
- 現在是第幾期？分期依據是甚麼？
- 是否有擴散？如果有，擴散到哪些位置？
- 還欠甚麼檢查才可決定治療？
- 需要做哪些 biomarker 或基因檢測？
- 治療目標是根治、減低復發、控制病情，還是紓緩症狀？
- 第一線治療建議是甚麼？有沒有其他合理選擇？
- 若我想尋求第二意見，應帶哪些資料？

## 關於治療選擇

- 為甚麼建議這個治療，而不是另一個方案？
- 治療成功的指標是甚麼？腫瘤縮小、指數下降、症狀改善，還是延長無惡化時間？
- 治療需要多久？每次相隔多久？
- 甚麼時候會評估療效？
- 如果第一個方案無效，下一步有甚麼選擇？
- 有沒有臨床試驗適合我？
- 治療會否影響生育、性功能、工作、駕駛或照顧家人？

## 化療前問題

- 化療藥物名稱是甚麼？每種藥的作用是甚麼？
- 總共幾多個週期？每個週期幾多日？
- 是門診、日間治療還是需要住院？
- 常見副作用有哪些？哪些副作用要即時求醫？
- 會否脫髮？有沒有頭皮冷卻或其他選擇？
- 需要預先放置輸液港或 PICC 嗎？
- 治療期間可以打疫苗、看牙醫、旅行或工作嗎？
- 發燒時應打哪個電話？是否需要直接去急症室？

## 電療前問題

- 電療目標是根治、術後輔助、止痛，還是控制局部症狀？
- 總共需要幾多次？每日一次還是其他安排？
- 照射範圍包括哪些器官？可能有哪些短期和長期副作用？
- 治療前是否需要定位、面罩、模具或呼吸控制？
- 照射位置皮膚應怎樣護理？
- 可否同時化療？如果可以，副作用會否增加？
- 甚麼症狀表示要暫停或通知醫護？

## 標靶或免疫治療前問題

- 我的 biomarker 是否支持這種治療？
- 藥物是口服、針劑還是輸液？需要多久一次？
- 需要用多久？甚麼情況會停藥或轉藥？
- 常見副作用是甚麼？例如皮疹、腹瀉、肝炎、肺炎、甲狀腺問題。
- 有哪些食物或藥物要避免？例如西柚、聖約翰草、某些中藥或保健品。
- 是否需要定期驗血、驗尿、心電圖或掃描？
- 如果忘記食藥，應怎樣處理？
- 是否屬自費藥？可否申請撒瑪利亞基金或關愛基金？

## 手術前問題

- 手術目標是根治、減低腫瘤負荷，還是取得組織診斷？
- 手術會切除哪些器官或組織？
- 有沒有微創、機械臂或開刀選擇？分別是甚麼？
- 預計住院幾多日？復原需時多久？
- 主要風險是甚麼？出血、感染、漏、神經損傷、造口？
- 術後是否仍需要化療、電療、標靶或免疫治療？
- 手術後的病理報告會提供哪些關鍵資料？
- 若我想保留病理切片或蠟塊作基因檢測，應怎樣安排？

## 關於費用和資助

- 這個治療在公立是否屬標準治療、自費項目，還是可申請資助？
- 如果在私家做，預計每個月費用大約是多少？
- 有沒有較便宜但醫學上仍合理的替代方案？
- 應何時找醫務社工？
- 若需要保險預先批核，診所可否提供治療計劃書和費用估算？

## 覆診前要帶甚麼

- 最新影像報告和 DICOM 光碟。
- 病理報告、基因檢測報告。
- 所有藥物、保健品、中藥清單。
- 最近血液報告。
- 症狀紀錄：痛楚、發燒、體重、胃口、大小便、皮疹、麻痺。
- 保險問題和財政限制。
- 家人或照顧者，幫忙記錄。

## 覆診後要確認

離開診症室前，請確認以下事項：

- 下一步檢查或治療是甚麼？
- 誰負責安排？大約何時收到電話？
- 如果等不到通知，應打哪個電話追問？
- 甚麼症狀需要立即求醫？
- 下一次覆診日期。

官方癌症網上資源中心亦提醒，覆診前記錄身體狀況和症狀，有助醫護作出判斷。

# 癌症診斷服務資助

在香港公立醫院排期進行影像檢查（如PET或MRI）通常需要較長時間。為了讓病人能盡快確定病情並開始治療，可以善用「見善計劃」。

### 什麼是「見善計劃」？
**「見善計劃」(Jian Shan Fund)** 旨在為合資格的癌症病人提供一次性資助，讓他們在私家醫療機構進行檢查，並帶著報告快速返回公營系統接受治療。

### 資助範圍
該計劃涵蓋以下關鍵的診斷項目：
* **初步腫瘤診斷 (Initial Oncology Diagnosis)**
* **電腦斷層掃描 (CT Scan)**
* **磁力共振掃描 (MRI)**
* **內窺鏡檢查 (Endoscopy)**
* **正電子掃描 (PET Scan)**



### 申請條件與目標
* **目標對象**：需要在公立醫院排隊但希望加快診斷速度的合資格癌症病人。
* **目的**：縮短等待時間，儘早確定癌症的分期與嚴重程度，從而獲得及時的醫療照顧。

> **小貼士**：由於檢查項目需由醫生推薦，建議在公立醫院專科門診諮詢時，主動向醫生查詢是否符合申請此計劃的資格。

# 癌症治療保險與自費預算指南

癌症治療的財務壓力，往往不只來自手術或住院。很多費用發生在門診、日間治療、藥物、掃描、基因檢測和副作用處理。開始私家治療前，最重要是先知道自己的保險真正包甚麼。

本頁不是保險建議，而是幫你準備與保險公司、醫生和醫務社工溝通時要問的問題。

## 先做三件事

1. 找出所有保單：個人醫保、公司醫療、危疾、癌症保、團體保險。
2. 致電保險公司，確認癌症相關保障，並要求書面紀錄或電郵。
3. 把保險限制告訴醫生，讓醫生知道你的現實選項。

## 癌症費用常見分類

| 類別 | 例子 | 常見問題 |
| --- | --- | --- |
| 診斷檢查 | CT、MRI、PET-CT、內窺鏡、切片 | 是否需住院才受保？是否需預先批核？ |
| 病理與基因檢測 | IHC、PD-L1、NGS、HER2、EGFR | 是否屬化驗？是否有上限？ |
| 手術 | 外科醫生、麻醉、手術室、病房 | 醫生費和醫院費是否分開計？ |
| 化療 | 門診或日間治療藥物 | 是否包門診化療？有沒有每次或每年上限？ |
| 標靶/免疫治療 | 口服藥、針劑、輸液 | 是藥物保障、癌症保障還是自費？ |
| 電療 | 定位、計劃、每日照射 | 是否包整個療程？有沒有技術限制？ |
| 副作用處理 | 入院、抗生素、輸血、止嘔藥 | 併發症是否當作癌症治療一部分？ |
| 康復與支援 | 物理治療、營養師、心理輔導 | 多數保單限制較多，要逐項問。 |

## 問保險公司的問題

請盡量逐條問，並記下日期、職員姓名和參考編號。

- 我的保單是否保障癌症門診治療？
- 化療、標靶藥、免疫治療是否需要住院才受保？
- 日間治療中心的輸液是否受保？
- PET-CT、MRI、CT 是否需要醫生信和預先批核？
- 基因檢測、NGS、PD-L1、HER2、EGFR 等是否受保？
- 口服標靶藥是在藥物保障、門診保障，還是完全不包？
- 有沒有每年上限、每次上限、終身上限？
- 有沒有墊底費、共同保險、自付比例？
- 指定網絡醫生或醫院會否影響賠償？
- 如果由公立轉私家，或私家轉公立，索償有沒有影響？
- 治療前是否需要 guarantee letter 或 pre-authorization？
- 需要哪些文件才可索償？正本收據、診斷證明、病理報告、藥物明細？

## 問私家醫生或診所的問題

- 請提供書面治療計劃和費用估算。
- 每個週期或每個月大約費用是多少？
- 費用是否包括醫生診金、藥費、輸液費、抽血、掃描和處理副作用？
- 如果出現發燒或感染，需要入院時大概費用如何？
- 可否先向保險公司申請預先批核？
- 如果中途轉回公立，如何取得完整治療紀錄？

## 問公立醫院或醫務社工的問題

- 這項藥物或治療是否屬自費項目？
- 是否可申請撒瑪利亞基金、關愛基金或醫療費用減免？
- 何時應開始申請？需要哪些文件？
- 申請批核前自行購買藥物，之後能否追回？
- 如果家庭資產或收入有特殊情況，應怎樣向社工解釋？

醫院管理局的撒瑪利亞基金和關愛基金醫療援助計劃，通常需要符合臨床條件和經濟審查。申請前應向主診醫生和醫務社工確認。

## 財務決策小貼士

- 把錢優先用在會影響治療決定的檢查，例如分期影像、病理、必要 biomarker。
- 不要因為保險快到期或焦急，就倉促開始未理解清楚的昂貴治療。
- 若保額有限，應及早建立公立跟進，避免私家費用突然超出承受能力。
- 每次付款前問清楚是否有替代方案、是否可延遲到公立、是否可申請資助。
- 保留所有收據、醫生信、藥物明細和報告。

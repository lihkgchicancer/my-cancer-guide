香港癌症治療：公立與私家醫院對比

在香港確診癌症後，病人通常面臨選擇在公立醫院（醫管局 HA）還是私家醫院治療。

1. 公立醫院 (Hospital Authority)​

  • 優點: 費用低廉（大部分檢查及手術有資助）、醫療團隊經驗豐富、設有撒瑪利亞基金資助昂貴藥物。
  • 缺點: 輪候時間較長（特別是 PET-CT、MRI 等檢查）、醫生每次可能不同、病房較擁擠。
  • 流程: 經私家醫生或專科門診轉介 -> 提交轉介信至聯網腫瘤科 -> 等待第一次見醫生（排期）。

2. 私家醫院 (Private Sector)​

  • 優點: 快速（通常一週內可開始治療）、環境舒適、可指定醫生、治療方案更靈活。
  • 缺點: 費用高昂（需考慮保險保額）、部分複雜併發症最終仍需轉回公立醫院處理。

小貼士: 許多病人會選擇「雙軌制」，即在私家醫院進行初期快速檢查（如切片、影像），再帶同報告回公立醫院排期治療。
